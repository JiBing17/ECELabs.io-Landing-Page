Why Use MyFPGA?

MyFPGA offers a powerful platform for digital design, enabling students and educators to create, simulate, and test their FPGA designs entirely online.

Whether you're learning digital logic, working on embedded systems, or designing complex hardware, MyFPGA provides an easy-to-use interface with advanced tools to help you succeed.

Sign up today and start your journey with MyFPGA!

What is MyFPGA?

MyFPGA is an online digital design platform based on Field-Programmable-Gate-Array (FPGA) and web technologies initially created by Junfei Li in 2016. It has been used by students and instructors for various courses as follows:

ECE Summer Camp — MS and HS outreach program introducing digital logic and hardware programming

Using a System-on-Chip (SoC) FPGA, including a hard processor interfaced with the FPGA fabric, the platform can also be used in other related courses such as:

  • Computer Organization and Microprocessors — Computer systems course featuring computer organization, ISA, assembly/C/C++/Python programming, I/O interfacing, Embedded Systems, Hardware, and Software Co-Designs

FAQ

Q1. Is this website safe for me?

Yes. The website uses Google sign-in for authentication. No password or personal information will be used. It uses the https protocol to secure communication between your browser and the server. Also see the answer to Q2.

Q2. Why did I get a privacy error and not secure warning?

This is because the server uses a self-signed certificate. While it is valid, it is not recognized by the browser. Follow the tutorial on how to install the certificate to your browser.

Q3. Why did I receive 'Unauthorized user! Redirected to log in page!'?

Please be assured that you are already properly registered with the website if you can see this page. The message only means that you are not recognized by the server at that moment. There are a few reasons: 1) You have been on the server for over 2 hours and you need to re-login. 2) The server was down and you need to re-login when it restarts. 3) The web service is reserved by a class and you will need to wait after the class is over. In this case, there will be an announcement on the login page on when the reservation will start and end.

Q4. Where are my design files?

On MyFPGA, you may find three storage places for your designs as follows: 1) Your local computer hard drive. This is the destination for Download or source for Import. 2) Your local browser storage. This is usually invisible to the user. It is used internally to save and load design automatically for you when you switch between pages or when you click on Save. 3) The remote server. Each user has a dedicated and unique directory to store files on the server. Designs will be uploaded from the browser to the server automatically when you Build the designs or when you click on Upload. They can then be retrieved into the browser from the File folder on the upper right section of the design pages.

Xilinx University Program hosted a Workshop entitled Integrating Compute Acceleration into Embedded System Design Using Vitic at the 2021 Embedded Systems Week Conference. Xilinx software tools were run from AWS and results designs were verified using remote PYNQ-Z2 boards from ECELabs, including Lab 2 on Embedded Systems, and Lab 3 on PYNQ Framework.

3rd Annual ECE Virtual Camp.

May 12, 2021

Xilinx University Program hosted a Workshop entitled From RTL to Compute Acceleration using Vitis and Cloud Computing at the 2021 IEEE International Symposium On Field-Programmable Custom Computing Machines (FCCM 2021). About 40 participants from around the world attended the live zoom sessions. During the first day of the two-day workshop, AWS was used to run Vivado and 20 PYNQ Z2 boards from ECELabs were used to validate the resulting designs of three labs, including Lab 1 on RTL, Lab 2 on Embedded Systems, and Lab 3 on PYNQ Framework.

We had 19 participants from Grand Valley State University, Morgan State University, New York University, Rensselaer Polytechnic Institute, Rice University, Texas A&M College Station, Texas State University, Texas A&M Kingsville, University of Massachusetts Lowell, University of Houston, University of Texas at Arlington, University of Texas at Austin, University of Texas at Dallas, University of Texas at San Antonio. 20 Basys3 boards were available for use.
